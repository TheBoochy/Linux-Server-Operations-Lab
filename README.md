# Linux Server Operations Lab

## Overview

Linux Server Operations Lab is a voluntary sysadmin portfolio project focused on practical Linux server administration.

The project demonstrates basic Linux operations skills, including server setup, SSH administration, users and groups, sudo access, firewall configuration, systemd service management, log review, backup scripting, scheduled jobs, monitoring and troubleshooting.

## Scenario

A small organization needs a Linux operations server that can be administered securely and documented properly.

The goal of this lab is to build, configure, test and document a Linux server environment using realistic sysadmin tasks.
